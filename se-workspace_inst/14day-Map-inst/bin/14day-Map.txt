- Iterator 인터페이스 
  review : interface 란 사용하는 측과 구현한 측과의 약속 
  			( 다중상속의 장점을 지원 -> 다양한 계층구조 형성을 통한 다형성 지원 ) 
  			
   java.util.Collection 계열의 interface 와 class 는 다양한 자료구조의 집합 
   다양한 데이터를 효과적으로 관리하기 위한 알고리즘을 미리 정의해 재사용하게 한다 
   
   Collection interface 의 자식 interface 는 Set 과 List 인터페이스가 있고 
   이를 구현한 많은 구현 클래스가 존재한다 
   
   컬렉션 계열의 많은 구현 클래스들이 담고 있는 자료를 단일한 방식으로 반복 , 열거하여 
   요소를 확인할 수 있는 것이 Iterator 인터페이스다. 
   
   Iterator interface -> 컬렉션 계열 구현체들의 요소를 단일한 방식으로 반복,열거하는
   							방법을 지원하는 인터페이스
